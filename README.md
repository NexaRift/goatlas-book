# GoAtlas Book

> A free, open source book to learn the Go programming language,
> written in Brazilian Portuguese and made for developers who already
> have basic programming logic knowledge.

## About

GoAtlas is not just another Go tutorial. It is a structured, didactic,
and community-driven book that guides you through the Go language from
the ground up — covering fundamentals, intermediate concepts, and
advanced topics in a clear and approachable way.

The book is written in Brazilian Portuguese because quality Go learning
resources in Portuguese are still scarce. We want to change that.

## What You Will Learn

- Core language fundamentals: variables, types, functions, and control flow
- Structs, interfaces, and Go's approach to object-oriented design
- Error handling the Go way
- Concurrency with goroutines and channels
- Generics, context, and testing
