# GoAtlas Book

> Um livro open source para aprender Go, escrito em Português Brasileiro.
> An open source book to learn Go, written in Brazilian Portuguese.

## Sobre / About

O GoAtlas é um livro voltado para quem já tem noções básicas de lógica
de programação e quer aprender Go do zero, de forma didática e direta.

GoAtlas is a book aimed at those who already have basic programming logic
knowledge and want to learn Go from scratch.
